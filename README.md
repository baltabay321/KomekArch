# KomekArch - Сайт для изучения компьютерной архитектуры

## Описание
Интерактивный образовательный сайт для изучения основ компьютерной архитектуры с помощью ИИ-ассистента.

## Технологический стек
- Next.js 13+
- React
- TypeScript
- Tailwind CSS
- API для интеграции с ИИ

## Установка и запуск

### Требования
- Node.js 16+
- npm или yarn

### Шаги установки
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/baltabay321/KomekArch.git
   cd KomekArch
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите сервер разработки:
   ```bash
   npm run dev
   ```
4. Откройте [http://localhost:3000](http://localhost:3000) в браузере
